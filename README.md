# BouncingPlus
<img width="1269" height="690" alt="menu_img" src="https://github.com/user-attachments/assets/ce99193c-008a-43f2-9443-949eb9dd5973" />
BouncingPlus is an advanced C++ remake of BouncingBallz, a game originally made in Python. Includes more enemy types, smarter enemy AI, custom weapons & levels, sfx & nusic, and much more.

## How To Play
You can download the latest release from GitHub or itch.io, or play the game in your web browser on itch.io.
itch.io link: https://rolpon.itch.io/bouncingplus
GitHub link: https://github.com/rolpon-cpp/BouncingPlus
